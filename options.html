LinkedIn Job Analyzer Settings

Configure your AI service provider to enable job summary generation.

AI Service Configuration

OpenAI: Get your API key from OpenAI Platform
Anthropic: Get your API key from Anthropic Console
Your API key is stored locally and never shared.

How to Use

  1. Navigate to any LinkedIn job posting
  2. Click the extension icon in your browser toolbar
  3. Choose between predefined format or custom format
  4. Click "Generate Summary" to get an AI-powered analysis
